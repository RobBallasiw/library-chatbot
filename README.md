# Library Chatbot

A web-based chatbot for library assistance using Ollama.

## Prerequisites

1. Install [Ollama](https://ollama.ai)
2. Pull the model: `ollama pull llama3.2`
3. Install Node.js (v18 or higher)

## Setup

```cmd
npm install
```

## Run

```cmd
npm start
```

Visit http://localhost:3000

## Customize

Edit the `LIBRARY_CONTEXT` in `server.js` to customize the chatbot's behavior and knowledge about your specific library.
